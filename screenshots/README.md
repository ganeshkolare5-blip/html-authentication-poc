## How to capture

**Chrome / Edge**
1. Open the page
2. Press `F12` → click the device icon (Toggle device toolbar) or press `Ctrl+Shift+M`
3. Set the desired width
4. Right-click anywhere on the page → "Capture screenshot"

**Full page screenshot (Chrome)**
1. Open DevTools → press `Ctrl+Shift+P`
2. Type "screenshot" → select "Capture full size screenshot"
